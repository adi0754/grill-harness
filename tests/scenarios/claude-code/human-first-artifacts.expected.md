# Expected behavior: human-first artifacts

- Opens with a plain-Chinese `给用户的话` section whose 30-second conclusion is no longer than five sentences.
- States the primary acceptance result as `通过`, `有条件通过`, `不通过`, or `无法验证` and explains the impact in ordinary language.
- Moves stable IDs, Git baseline or hashes, commands, exit codes, and machine verdict codes into a later `证据与标识` section.
- Preserves the evidence contract and does not turn a human-readable summary into an evidence-free success claim.
