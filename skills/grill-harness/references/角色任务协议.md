# 角色任务协议

每个正式角色只接收一份本地任务包。任务包必须自包含，但不得附带原始聊天历史；以稳定 ID、已批准产物和真实仓库代替会话记忆。

本地启动提示词保持简短：

> 读取任务包 `<任务包绝对路径>`，检查真实仓库，只执行授权范围，运行要求的验证，并把报告写回任务包指定路径；遇到停止条件立即停止并报告。

Codex 或 Claude Code 的新本地会话均可使用该提示词。启动提示词只负责要求读取任务包、检查真实仓库、执行任务和写回报告，不复制任务正文，不指定模型品牌，不生成网页模型便携提示词。

实施者不能最终批准自己；审查者必须读完整文件和真实 diff；验收必须使用新的本地 Agent 会话。正式角色只写报告，主会话核验证据后更新状态。
