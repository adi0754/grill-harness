# Claude Code 运行时

在新的本地 Claude Code 会话中发送 `角色任务协议.md` 生成的短启动提示词。任务包路径、产品项目目录和报告路径都必须是绝对路径；会话从任务包和真实仓库重建上下文，不依赖原聊天历史。

允许使用任务所需且已安装的 Skills。必需能力缺失时停止并报告，不静默模拟或安装。任务包未授权时，不创建/切换分支或 worktree，不提交、推送、合并、发布，也不执行破坏性 Git 操作。

正式角色只修改授权产品路径和指定的 `~/.grill-harness/` 报告路径。它可以报告结果，但不能直接批准用户门禁或把 Harness 状态标为通过。

完成时返回报告绝对路径、当前 Git 基线和验证结论。没有真实命令证据时使用“无法验证”，不能声称完成。
